# 隐私政策

版本：1.0
生效日期：2026年4月29日

## 1. 零数据收集
本应用不收集、存储或传输任何个人数据。

- **无网络请求**：所有处理在浏览器本地完成，不向任何服务器发送数据。
- **无遥测**：我们不追踪使用情况、错误或分析数据。
- **无 Cookie**：不使用追踪 Cookie 或持久化标识符。

## 2. 本地存储
应用仅在浏览器的 `localStorage` 中存储用户偏好设置（主题、语言、拼接参数等）。这些数据不会离开你的设备。

## 3. 图片处理
所有图片完全在浏览器内存中处理。没有图片数据被上传、分享或保存到任何外部服务。

## 4. 源代码透明度
作为开源项目，你可以随时审计源代码以验证上述声明的真实性。

## 5. 联系方式
如有隐私相关问题，请在 GitHub 上提交 Issue。

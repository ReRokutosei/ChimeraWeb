# Privacy Policy

Version: 1.0
Effective Date: April 29, 2026

## 1. Zero Data Collection
This application does not collect, store, or transmit any personal data.

- **No network requests**: All processing happens locally in your browser. No data is sent to any server.
- **No telemetry**: We do not track usage, errors, or analytics.
- **No cookies**: No tracking cookies or persistent identifiers are used.

## 2. Local Storage
The app stores only user preferences (theme, language, stitching parameters) in your browser's `localStorage`. This data never leaves your device.

## 3. Image Processing
All images are processed entirely in memory within your browser. No image data is uploaded, shared, or saved to any external service.

## 4. Source Code Transparency
As an open-source project, you can audit the source code at any time to verify the above statements.

## 5. Contact
For privacy-related questions, please open an issue on GitHub.
